xrpl.services Generic Backend


How do I use the Xaman Generic Backend?

The xrpl.services Generic Backend has a very simple API with some really cool features. Most of the "hard work" is done automatically by the backend. (Setting your destination address, setting up Xaman push notifications, and so on ...)

XUMM Buttons

The easy-to-integrate Xrpl.Services Buttons offer you an very simple way to add a Xaman button into your website. As example: for donations/tips/payments but also for offering exclusive content (after payment)

How to use the Xrpl.Services Buttons:

You have to be whitelisted for the xrpl.services Generic Backend (see steps above).
For adding one or more of the buttons, follow the detailed steps below.

Add the following script into the <head> section of your website:
<script defer src="https://pay.xrpl.services/payViaXumm.js"></script>

Now you can add a <div> tag anywhere in your website where you want to have a pay button.

The <div> tag must have the attribute: class="xumm-pay-button":
<div class="xumm-pay-button"></div>

There is also a button without a surrounding border:
<div class="xumm-pay-button-no-border"></div>

The Xaman Generic Pay Button will be inserted into the <div> element.
If you want to change the size of the button, just change the size of the surrounding <div>.

 Add the following script into the <head> section of your website:
<script defer src="https://donate.xrpl.services/donateViaXumm.js"></script>
